# Device Attributes Schema

To classify a medical device, the system must extract and interpret key attributes.

## Core Attributes

### Intended Use
- Diagnostic
- Therapeutic
- Monitoring
- Support or sustain life

### Anatomical Location
- External
- Internal
- Implantable

### Duration of Contact
- Transient (< 60 minutes)
- Short-term (< 30 days)
- Long-term (> 30 days)

### Invasiveness
- Non-invasive
- Minimally invasive
- Invasive

### Materials
- Metal
- Polymer
- Biological
- Composite

### Sterility
- Sterile
- Non-sterile

These attributes form the basis for classification logic and risk mapping.
